# スペースインベーダー ゲーム仕様書

## 概要

オーソドックスなスペースインベーダーのクローンゲームです。80年代のアーケードゲームを意識したレトロなデザインを採用しています。

## ゲームの特徴

### 登場するオブジェクト

| オブジェクト | 説明 |
|-------------|------|
| インベーダー | 画面上部から左右移動しつつ降りてくる敵。縦5段×横11列の隊列が初期状態 |
| バリア | インベーダーや自機の弾が当たると削れる防御壁。4つ配置 |
| 自機 | プレイヤーが操作する砲台 |
| UFO | 一定時間経過後に画面上部を端から端へ飛ぶ。撃ち落とすと300点のボーナス |

### インベーダーの種類とスコア

| 段（上から） | 色 | スコア |
|-------------|-----|--------|
| 1段目 | 赤 | 50点 |
| 2段目 | オレンジ | 40点 |
| 3段目 | 黄 | 30点 |
| 4段目 | 緑 | 20点 |
| 5段目 | シアン | 10点 |

## 操作方法

| キー | 動作 |
|------|------|
| ← → | 自機を左右に移動 |
| SPACE | 弾を発射 |
| S | ゲーム開始 |
| P | 一時停止/再開 |
| R | リセット |

## ゲームルール

1. プレイヤーは自機を操作してインベーダーを撃ち落とす
2. インベーダーが画面下部に到達するとゲームオーバー
3. バリアは一定回数の攻撃を受けると破壊される
4. UFOを撃ち落とすとボーナス得点（300点）
5. インベーダーを全て撃ち落とすと次のステージに進む
6. 自機は敵の弾に当たると残機が減り、残機がなくなるとゲームオーバー
7. 自機の弾は1発のみで、画面上部に到達するか他のオブジェクトにあたって消滅するまで次の弾を発射できない
8. ゲーム開始時に自機の残機は3つ
9. インベーダーの数が減るごとに移動速度が速くなる（最大10倍）

## 難易度システム

- レベルが上がるごとにインベーダーの弾の発射頻度が増加
- インベーダーの残り数に応じて移動速度が指数的に上昇

## サウンドシステム

Web Audio APIを使用した効果音:

- インベーダー移動音: 4音（75.9Hz, 66.2Hz, 62.2Hz, 57.7Hz）をループ
- 自機発射音: 下降する矩形波
- インベーダー撃破音: 下降する矩形波
- 自機被弾音: ホワイトノイズによる爆発音
- UFO出現音: LFOによるピロピロ音
- UFO撃破音: 下降するのこぎり波
- ゲームオーバー音: 下降する5音のメロディ

## UI要素

- スコア表示
- ハイスコア表示（ローカルストレージに保存）
- 残機表示
- レベル表示
- START/PAUSE/RESETボタン
- 操作説明
- 被弾時の赤い画面演出
- ゲームオーバーポップアップ
- ステージクリアポップアップ
- CRTブラウン管風のスキャンライン効果

## 技術仕様

- HTML5 Canvas による描画
- Web Audio API によるサウンド生成
- LocalStorage によるハイスコア保存
- レスポンシブデザイン（ビューポートの90%サイズで表示）
